# EPUB Samples

This folder is for storing sample EPUB files for testing the import functionality.

## Usage
1. Place `.epub` files here.
2. Use the "Import EPUB" button in the application to load them.

## Recommendations
- Use public domain books (e.g., from Project Gutenberg).
- Keep file sizes small (1-5 MB) for optimal performance.
- Valid EPUB 2.0 or 3.0 format.
